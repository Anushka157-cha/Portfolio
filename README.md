- **Hero Section** (Lines 75-90): Your name and description
- **About Me** (Lines 118-140): Your personal bio
- **Stats** (Lines 136-149): Update your numbers
- **Skills** (Lines 245-340): Add/remove skills

### Change Color Scheme
Edit `styles.css`:
```css
:root {
    --color-primary: #0EA5E9;      /* Sky blue */
    --color-secondary: #8B5CF6;    /* Purple */
    --color-accent: #06B6D4;       /* Cyan */
}
```

### Modify Typing Effect

```javascript
typingTexts: [
    'FullStack Developer',
    'AI Engineer',
    'Your Role Here'
]
```
